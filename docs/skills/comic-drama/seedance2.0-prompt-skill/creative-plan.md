# Seedance 2.0 创意企划：四大系列

> 基于 1.5W 即梦积分的创作计划，聚焦视觉冲击力和系列化传播。

---

## 系列一：「穿越万象」— 一镜到底穿越系列

**核心概念**：以"穿越"为母题，每集展现一种穿越类型，全程一镜到底不切镜头。系列化输出，适合连续发布形成系列IP。

**视觉风格**：每集两种风格的碰撞与过渡（如写实→水墨、现代→古风）

| 集数 | 穿越类型 | 核心画面 |
|------|----------|----------|
| 01 | 时间穿越 | 现代都市街头 → 穿过一扇旋转门 → 出来是唐朝长安万国来朝的街市 |
| 02 | 空间穿越 | 地球上仰望星空 → 镜头穿入星空 → 出来时已经站在火星红色荒漠上，回头看地球悬在天边 |
| 03 | 维度穿越 | 真实街头摄影画质 → 走进一面镜子 → 出来变成吉卜力/皮克斯风格的卡通世界 |
| 04 | 极限穿越 | 宇航员从空间站跳下 → 穿过大气层 → 穿过海面 → 穿过地壳 → 穿到地核 → 穿出去是黑洞的另一端 |

**关键Seedance能力**：一镜到底、风格过渡、运镜组合技

**建议时长**：每集15秒

**扩展方向**：
- 情感穿越：从悲伤的场景穿越到温暖的场景
- 文明穿越：从原始部落一路穿越到太空时代
- 尺度穿越：从原子结构穿越到宇宙尺度（Powers of Ten）

## 系列二：「太极生万物」— 东方玄幻创世

**核心概念**：以太极哲学为骨架，演绎从混沌到万物的东方创世过程。太极 → 两仪 → 四象 → 八卦 → 万物，一气呵成。

**视觉风格**：水墨 + 3D国漫CG混合渲染，东方美学极致表达

**叙事结构**（30秒，2段拼接）：

### 第1段（0-15秒）：混沌 → 太极 → 两仪

- **0-3秒**：一片混沌虚无的黑暗，微弱的光点在黑暗中旋转聚集
- **4-8秒**：光点汇聚成一个发光的太极图，缓缓旋转，水墨纹理在表面流动
- **9-12秒**：太极图裂开，黑白分离，化为日与月，天与地在画面中形成
- **13-15秒**：天地初开，大气中水墨云雾弥漫，为下一段做衔接

### 第2段（15-30秒）：四象 → 八卦 → 万物 → 归一

- **0-1秒**：桥接，云雾微动延续
- **1-5秒**：天地之间涌出四种元素——水流、火焰、山石、风暴（四象）
- **6-10秒**：八种力量交织汇聚，大地万物苏醒——山河成形、草木生长、瑞兽奔腾、仙鹤飞起
- **11-13秒**：镜头缓缓拉远，所有一切缩小
- **14-15秒**：万物凝聚回一个太极图，漂浮在宇宙星空之中，归于起点

**配乐方向**：古琴或箫的东方音色，从空灵到宏大再回归空灵

**关键Seedance能力**：史诗大制作模板、超长视频分段拼接、大气连贯声明

**扩展方向**：
- 五行版：金木水火土各自生成一段独立短片
- 二十四节气版：每个节气一个15秒短片
- 山海经版：以山海经为蓝本，生成异兽觉醒的画面

## 系列三：「时间扭曲者」— 超现实叙事短片

**核心概念**：以"时间异常"为切入点，展现现实中不可能发生的超现实场景。强故事感、强情绪感。

**视觉风格**：电影级写实质感，冷灰蓝色调，悬疑氛围

### 版本A：万物定格

> 主角在闹市中突然停住，所有人定格不动（子弹时间），只有他能移动。镜头360度环绕他，他触碰定格的雨滴、定格的飞鸟、悬停在空中的咖啡水花。

**情绪**：孤独 + 超脱 + 一丝不安

### 版本B：空城独行

> 主角醒来，空旷的城市，阳光照在无人的十字路口。他走过空无一人的街道、地铁、学校。最后在城市最高处俯瞰——远处地平线有一道光墙正在逼近。

**情绪**：孤寂 → 不安 → 紧迫

### 版本C：时间倒流

> 主角拥有倒转时间的能力。一杯摔碎的咖啡倒流回杯中、散落的花瓣倒飞回枝头、坠落的雨滴倒升回云端。美丽而忧伤。

**情绪**：温柔 + 遗憾 + 释然

**建议时长**：15秒单集 或 30秒两段式

**关键Seedance能力**：情绪演绎、运镜组合技、时间戳分镜精准控制

**扩展方向**：
- 时间加速者：触碰物体让它极速老化/生长
- 记忆穿行者：走进照片进入过去的记忆场景
- 平行选择者：同一个路口，左转和右转看到两个平行人生

## 系列四：「一滴水的宇宙」— 微观 × 哲学

**核心概念**：从微观潜入一滴水珠内部，发现里面是一个完整的宇宙，形成无限嵌套的递归结构。视觉震撼+哲学深度。

**视觉风格**：微距摄影 → 宇宙CG，两种极端尺度的碰撞

**叙事结构**（15秒，一镜到底）：

- **0-3秒**：微距特写，一片绿叶上的一滴露珠，晨光折射出彩虹
- **4-6秒**：镜头潜入水珠内部，水的折射纹理过渡为星云
- **7-10秒**：星云中出现行星，镜头靠近行星，表面有城市灯光
- **11-13秒**：进入城市，一个人站在阳台上，手中托着一滴水珠
- **14-15秒**：镜头再次潜入这滴水珠——暗示无限循环

**关键Seedance能力**：一镜到底、微观到宏观的连续尺度变换

**扩展方向**：
- 一粒沙的文明：沙粒中藏着一个沙漠文明
- 一片雪花的记忆：每片雪花的结晶结构里存储着一段记忆
- 瞳孔中的宇宙：从人的眼睛瞳孔深入，发现内心宇宙
